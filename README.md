# ZuckUSB
Replace the Windows USB connect/disconnect sounds with custom audio clips. (Defaults to "Mark Zuckerberg" and "No you're not" as a reference to this video: https://www.youtube.com/shorts/3ETts3FGIgM)

# Instructions

1. Right click the installer and Run as administrator.

<img width="618" height="547" alt="image" src="https://github.com/user-attachments/assets/a9de15bd-f984-42de-a2d0-957ea7a8322a" />
<hr>

2. The installed sounds will both play once and a success message will be displayed in the console.

<img width="1101" height="637" alt="image" src="https://github.com/user-attachments/assets/b1a8c3bc-c3af-4177-a58c-22accc0efc88" />
<hr>

To revert the changes, run the revert file as administrator to revert back to the default sounds and to automatically delete the added audio files.

<img width="1101" height="637" alt="image" src="https://github.com/user-attachments/assets/77b852df-0b36-4e55-a775-77d072a910e5" 
/>

Or run the revert file without administrator permissions to revert back to the default sounds, but the added audio files won't be removed from your disk.
